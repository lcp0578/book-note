- 字面量
	- 字面量是直接出现在Java源码中的值，包括整数、浮点数、单引号中的单个字符、双引号中的字符串，以及保留字 `true`、`false`和`null`。
- 位运算符和位移运算符
	- 按位补码（`~`）
	- 位与（`&`）
	- 位或（`|`）
	- 异位或（`^`）
	- 左移（`<<`）
	- 带符号右移（`>>`）
	- 不带符号右移（`>>>`）
