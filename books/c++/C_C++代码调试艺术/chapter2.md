## 第2章 Visual C++调试基本功能
- Visual C++（简称为VC） 是微软公司的C++开发工具，具有吉成开发环境，可提供编辑、编译、调试C\C++等标称语言的功能。
- VC对C语言和C++语言的支持紧密结合C和C++标准
	- VC2015基本上完整地支持C99标准
	- VC2017版本支持C++17
	- VC2019版本支持C++20标准大部分核心功能以及几乎全部C++20标准库。
- 断点管理
- 调试执行
- 监视/快速监视
- 内存查看
- 通过“局部变量”窗口和“自动”窗口查看变量
- 调用堆栈
- 多线程管理
- 查看错误码
- 调试宏Assert使用
- 其他
